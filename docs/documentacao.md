# `docs/` - Documentacao do projeto

Materiais de apoio: apresentacao, guias e perguntas/respostas para a banca.

## Arquivos

| Arquivo | Como usar |
|---------|-----------|
| [[apresentacao]] | Apresentacao em Markdown (Marp). Slides para a banca. |
| `apresentacao.html` | Mesma apresentacao em HTML standalone. Abre no navegador, sem extensao. |
| [[perguntas-e-respostas]] | Guia de estudo com perguntas que a banca pode fazer. |

## Mapa de navegacao

A partir do [[../README|README raiz]] voce navega para:

- [[apresentacao]] - apresentacao em Markdown (Marp)
- `apresentacao.html` - mesma apresentacao standalone
- [[perguntas-e-respostas]] - guia de estudo
- [[../src/visao-geral|src/]] - codigo fonte
- [[../sql/scripts-sql|sql/]] - schema do banco

## Como abrir

- **Apresentacao slides**: instale a extensao Marp no VS Code/Kiro e abra
  `apresentacao.md`. Para exportar PDF: `Marp: Export Slide Deck...`.
- **Apresentacao HTML**: duplo clique em `apresentacao.html`. Use setas
  para navegar.
- **Perguntas e respostas**: leitura direta no Obsidian/preview do MD.
